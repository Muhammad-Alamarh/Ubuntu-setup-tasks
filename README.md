# أشياء يجب القيام بها بعد تثبيت Ubuntu:
## تحديث النظام:
Applications menu > Software updater > Install now

**Terminal Command:**  
```bash
sudo apt update && sudo apt upgrade
```
## أضافة مصادر اخرى للبرامج:
Appication menu > software & updates > other software
